### 🕹 Про проєкт
Це реконструкція легендарної гри **Battle City** (1990), створена з нуля на рушії **Phaser 3**. Метою було не просто скопіювати механіку, а відчути архітектуру класичного геймдизайну через сучасний JavaScript.

### 🚀 Технічні особливості
*   **Engine:** Phaser 3 (Arcade Physics).
*   **Architecture:** Модульна структура (Managers для колізій, будівництва рівнів та стейджів).
*   **Level Design:** Система завантаження рівнів через матриці (31x25), що дозволяє легко створювати нові карти.
*   **Environment:** Реалізована інерція на льоду, непрохідна вода для танків та прозорість для куль.
*   **Adaptive UI:** Унікальна верстка «Телевізор» на CSS Grid + Aspect Ratio, що ідеально вписує гру в будь-який монітор.

### 🛠 Як запустити
1. Клонувати репозиторій.
2. `npm install`
3. `npm run dev`
